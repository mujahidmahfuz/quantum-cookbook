What's new, and an optional way to track it

This is not a real account — there's no password, no server, and it doesn't sync between devices or browsers. It's just a name saved in this browser's local storage, used only to say "welcome back" and to mark which updates below you've already seen. Skip it entirely if you'd rather not bother — every chapter works exactly the same either way.

Want real email notifications instead of this local tracker? That's not wired up yet — it would need a third-party mailing-list service, since this site has no backend of its own. Get in touch if that's something you'd actually use.
